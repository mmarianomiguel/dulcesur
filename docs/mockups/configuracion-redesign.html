Cambios clave en esta propuesta

  • 3 grupos lógicos (Negocio / Tienda Online / Sistema) reemplazan 8 páginas sueltas
  • Footer unificado — eliminada la duplicación entre Tienda y la página Footer separada
  • Pagos movido a Negocio — junto con Facturación e Impresión donde tiene sentido
  • Página de Inicio dentro de Tienda — ya no es una sección separada en el sidebar
  • Apariencia dentro de Sistema — agrupa customización técnica del panel
  • Navegación secundaria — sidebar de config con secciones agrupadas y labels de grupo
  • Sin tabs dentro de cada sección — cada subsección es una página propia con scroll, más limpio
  • 1 URL por sección — /configuracion/negocio/empresa, /configuracion/tienda/footer, etc.
